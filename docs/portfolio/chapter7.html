Chapter 7: Classification & Clustering Analysis

In this chapter, we will look at clustering and multiple classification algoritms. Instead of identifying a coast by hand, it will be interesting to see whether there can be algorithmicly determined to which coast a song belongs. First, we will start with a small feature selection section. Then, we try K-Means clustering. After that, we try KNN, A single descision tree and Random Forrest.

Feature Selection

In chapter two we saw that a lot of the distributions of features were the same for both coasts. So, to remove nois for both clustering and classification, we removed Key, Time Signature, Energy, Liveness, Mode, Loudness and Duration.

K-Means Clustering

K-Means clustering is an unsupervised learning technique that partitions the data into k distinct groups based on feature similarity. As we are trying to find out whether K-Means can distinguish the two coasts, we set the K to 2.

Clustering Visualization

This is the with PCA visualised result of K-Means clustering for E.C. and W.C. Before looking at metrics, you can see that the clustering did not do well on disecting east and west. The blue cluster is quite small in relation to the orange cluster and shapes differ throughout. Also in the orange cluster is a lot of differentiation.

The Silhouette Score ranges from -1 to 1, where higher values indicate better-separated clusters. Thus, the silhoutte score achieved indicates weak structure. Or, in other words, there is some structure in the data, but not a lot.

K-Nearest Neighbors (KNN) Classification

K-Nearest Neighbors is a supervised classification algorithm that predicts a song’s region by finding the k most similar songs in the training data and using majority voting. Through 5 fold cross-validation, we identified the optimal value of k that balances bias and variance.

Decision Tree Classification

Decision Trees provide an interpretable classification model by recursively partitioning the feature space based on feature values. This model’s main advantage is explainability: we can visualize exactly which features and thresholds the algorithm uses to make predictions, and we can extract feature importance scores.

Random Forest Classification

Random Forest is an ensemble learning method that builds multiple decision trees and aggregates their predictions. By combining many trees trained on random subsets of data and features, it reduces overfitting and captures complex non-linear patterns that single trees miss. This is particularly valuable for our hip-hop classification task, where subtle production style interactions matter.
